# Eagle-sensor-libraries
Custom EAGLE libraries containing symbols, footprints, and device packages for popular sensors.
This repository contains custom EAGLE CAD libraries for commonly used sensors. It includes schematic symbols, PCB footprints, and properly mapped device packages to simplify PCB design and ensure accurate hardware implementation. These libraries are intended for students, hobbyists, and engineers working on electronic projects.
